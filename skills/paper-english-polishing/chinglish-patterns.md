## Category 1: Articles (the/a/an) — Most Frequent Error

Chinese has no articles. Native Chinese speakers systematically omit or misuse them.

### 1a. Missing Definite Article

| Wrong | Correct | Rule |
|-------|---------|------|
| "in network environment" | "in **the** network environment" | Specific environment being discussed |
| "method achieves best performance" | "**the** method achieves **the** best performance" | Specific method + superlative |
| "as shown in experiment" | "as shown in **the** experiment" | Referring to specific experiment |
| "result indicates that" | "**the** result indicates that" | Specific result being discussed |
| "with development of deep learning" | "with **the** development of deep learning" | Specific development |

### 1b. Missing Indefinite Article

| Wrong | Correct | Rule |
|-------|---------|------|
| "This is important contribution" | "This is **an** important contribution" | Countable noun needs article |
| "We propose novel framework" | "We propose **a** novel framework" | First mention, countable |
| "It serves as effective defense" | "It serves as **an** effective defense" | Countable noun |

### 1c. Unnecessary Article

| Wrong | Correct | Rule |
|-------|---------|------|
| "the Equation 1" | "Equation 1" or "Eq.~(1)" | Numbered references: no article |
| "as shown in the Fig. 1" | "as shown in Fig.~1" | Abbreviated + numbered: no article |
| "the Table 2 shows" | "Table 2 shows" | Numbered reference: no article |
| "the APT attacks are" (generic) | "APT attacks are" | Generic class: no article |
| "the deep learning has" (generic) | "Deep learning has" | Uncountable abstract: no article |

### 1d. Article with Specific Constructs

| Context | Rule | Example |
|---------|------|---------|
| "proposed method" | Use "the" | "**the** proposed method" (NOT "our proposed") |
| Proper nouns (method names) | No article | "HARP achieves" (NOT "the HARP") |
| Dataset names | No article | "SAGA v2 contains" (NOT "the SAGA v2") |
| "following section" | Use "the" | "in **the** following section" |
| "same" | Always "the" | "**the** same dataset" |

## Category 2: Uncountable/Plural Nouns

| Wrong | Correct | Why |
|-------|---------|-----|
| "researches" (noun) | "studies" / "research efforts" | "research" is uncountable |
| "datas" | "data" | Already plural (datum → data) |
| "informations" | "information" | Uncountable |
| "knowledges" | "knowledge" | Uncountable |
| "evidences" | "evidence" | Uncountable |
| "literatures" | "the literature" / "prior studies" | Uncountable in this sense |
| "an anomaly detection" | "anomaly detection" | Abstract concept, no article |
| "softwares" | "software" / "software tools" | Uncountable |
| "feedbacks" | "feedback" | Uncountable |
| "equipments" | "equipment" | Uncountable |

## Category 3: Verb Patterns

### 3a. Subject-Verb Agreement

| Wrong | Correct | Rule |
|-------|---------|------|
| "the data shows" | "the data **show**" or "the dataset **shows**" | Choose one house style and stay consistent |
| "the criteria is" | "the criteria **are**" | "criteria" is plural |
| "each of the methods achieve" | "each of the methods **achieves**" | "each" is singular |
| "a number of studies shows" | "a number of studies **show**" | "a number of" takes plural |
| "the number of attacks are" | "the number of attacks **is**" | "the number of" takes singular |

### 3b. Dangling Modifiers

| Wrong | Correct |
|-------|---------|
| "Using GRU, the accuracy is improved" | "Using GRU, **we improve** the accuracy" |
| "Based on the observation, a method is proposed" | "Based on the observation, **we propose** a method" |
| "By analyzing the data, several patterns were found" | "By analyzing the data, **we identified** several patterns" |
| "Compared with baseline, better results are obtained" | "Compared with **the** baseline, **HARP achieves** better results" |

**Rule**: The subject of the main clause must perform the action in the modifier.

### 3c. Chinese-Influenced Verb Choices

| Chinglish | Native English |
|-----------|---------------|
| "put forward a method" | "propose a method" |
| "carry out experiments" | "conduct experiments" |
| "make use of" | "use" / "employ" |
| "have a great influence on" | "significantly affect" |
| "give a description of" | "describe" |
| "make an improvement" | "improve" |
| "do research on" | "investigate" / "study" |
| "play an important role" | "is important for" / "contributes to" |

## Category 4: Prepositions

Chinese does not distinguish prepositions as finely as English.

| Wrong | Correct | Context |
|-------|---------|---------|
| "be composed by" | "be composed **of**" | Composition |
| "be familiar to sth" | "be familiar **with** sth" | Familiarity |
| "be similar with" | "be similar **to**" | Similarity |
| "different with" | "different **from**" | Difference |
| "superior than" | "superior **to**" | Comparison |
| "consist in" (wrong sense) | "consist **of**" | Composition |
| "result of" (cause) | "result **from**" / "result **in**" | Cause vs. effect |
| "contribute for" | "contribute **to**" | Contribution |
| "aim to do" vs "aim at" | "aim **to** + verb" / "aim **at** + noun" | Purpose |

## Category 5: Sentence-Level Patterns

### 5a. Redundant Constructions

| Redundant | Concise |
|-----------|---------|
| "in order to" | "to" |
| "due to the fact that" | "because" |
| "for the purpose of" | "to" / "for" |
| "a total of 100 samples" | "100 samples" |
| "whether or not" | "whether" |
| "the reason is because" | "the reason is that" / "because" |
| "it can be seen that" | (delete, state directly) |
| "it is worth mentioning that" | (delete, state directly) |
| "as is well known" | (delete, state the fact) |
| "in the field of" | "in" |
| "with respect to" | "for" / "regarding" |
| "in the case of" | "for" / "when" |
| "at the present time" | "currently" / "now" |
| "has the ability to" | "can" |
| "despite the fact that" | "although" |

### 5b. Redundant Twins (Choose ONE)

| Redundant Pair | Keep One |
|----------------|----------|
| "help and assist" | "help" or "assist" |
| "method and approach" | "method" or "approach" |
| "monitor and observe" | "monitor" or "observe" |
| "predict and forecast" | "predict" or "forecast" |
| "basic and fundamental" | "fundamental" |
| "first and foremost" | "first" |
| "various and different" | "various" |
| "each and every" | "each" or "every" |

### 5c. "respectively" Usage

| Wrong | Correct |
|-------|---------|
| "A and B are 0.9 and 0.8 respectively" | "A and B are 0.9 and 0.8**,** respectively" (comma required) |
| "We use A for X. We use B for Y respectively." | "We use A for X and B for Y, respectively." (must be in same sentence) |
| "The AUC is 0.99 respectively" (no pair) | "The AUC is 0.99" (delete "respectively" — only one item) |

## Category 6: Logical Connectors

Chinese tends to over-signal logical relationships with explicit connectors. English prefers fewer, more precise transitions.

### 6a. Over-Used Connectors (Reduce)

| Overused | When to Delete | When to Keep |
|----------|---------------|-------------|
| "However" | When contrast is already clear from content | When introducing genuinely unexpected contrast |
| "Therefore" | When causation is obvious | When summarizing a multi-step argument |
| "Furthermore" | When additive relationship is trivial | When adding genuinely new evidence |
| "Meanwhile" | Almost always wrong in academic English | Use "simultaneously" or restructure |

### 6b. False Friends

| Chinese-Influenced | Correct English |
|--------------------|----------------|
| "at the same time" (for contrast) | "however" / "in contrast" |
| "on the contrary" (for mere contrast) | "in contrast" / "by contrast" ("on the contrary" implies the opposite is true) |
| "especially" (starting sentence for emphasis) | "in particular" / "notably" |

## Category 7: Number and Measurement Expressions

| Wrong | Correct | Rule |
|-------|---------|------|
| "4 datasets were used" (sentence start) | "**Four** datasets were used" | Spell out numbers <10 at sentence start |
| "5 %" | "5\%" (LaTeX) | No space before % |
| "4554344" | "4,554,344" | Thousands separator |
| "0.997 and 0.9984" (same table) | Same decimal places in same table | Decimal consistency |
| "about 10%" | "approximately 10\%" | Use formal word |
| "bigger than" | "greater than" / "larger than" | Formal comparison |
| "2 times" | "twice" / "twofold" | Formal multiplier |

## Category 8: Chinese Punctuation (Zero Tolerance)

| Chinese | English | Action |
|---------|---------|--------|
| （ ） | ( ) | Replace |
| ， | , | Replace |
| 。 | . | Replace |
| ； | ; | Replace |
| ： | : | Replace |
| " " | `` '' (LaTeX) or " " | Replace |
| 、 | , | Replace |

**Every Chinese punctuation mark in an English paper is an error.**
