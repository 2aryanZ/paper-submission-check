1. Open paper-submission-check from the desktop shortcut or Start menu.
2. You will see a simple interface asking you to upload your paper file.
3. Click "Browse" or the upload button.
4. Locate the paper file on your computer. The software supports common formats like `.docx` and `.pdf`.
5. Select the file and click "Open".
6. Press the "Run Checks" button to start.
7. The software will go through various checks:
   - Document length  
   - Citation formatting  
   - Page layout  
   - Required sections (abstract, keywords, references)  
   - File format compatibility  
8. Results will show on the screen. Errors or warnings will be highlighted.
9. To fix issues, edit your paper file, then upload and check it again.
10. When your paper passes all checks, it is ready for submission.
